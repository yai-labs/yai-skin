# Code of Conduct

## Our Standards

- Be respectful and constructive.
- Focus discussion on technical outcomes and user value.
- Accept feedback and collaborate in good faith.
- Avoid harassment, insults, discrimination, and personal attacks.

## Scope

This policy applies to all repository interactions: issues, pull requests, discussions, and reviews.

## Enforcement

Report concerns to the maintainers listed in `SECURITY.md`.
Maintainers may edit, remove, or reject contributions that violate this code of conduct.
