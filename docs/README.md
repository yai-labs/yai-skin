# Docs

Lightweight documentation for `yai-skin`, aligned with the YAI ecosystem style and adapted to a CSS library scope.

## Core

- `architecture.md`: layer model, public contract, compatibility rules.
- `release.md`: versioning and release checklist.
- `governance.md`: contribution and policy model for this repo.
- `ci.md`: workflows and required status checks.
- `consumers.md`: integration contract for consumer repositories.

## Tooling

- `tooling/doxygen.md`: when Doxygen is useful, limits for CSS, and how to enable it.

## Generated

- `_generated/architecture-alignment.v1.json`: generated snapshot used by architecture-alignment checks.
