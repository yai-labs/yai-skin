# Tokens Layer

Design tokens are the canonical visual contract of `yai-skin`.

## Files

- `colors.semantic.css`: semantic color aliases and state tokens.
- `colors.yai-dna.css`: brand/core palette primitives.
- `spacing.css`: spacing scale and sizing tokens.
- `typography.css`: typography scale and text tokens.
- `themes/`: theme-level overrides.

## Rules

- Additive changes should preserve backward compatibility.
- Breaking token renames/removals require a major version bump.
