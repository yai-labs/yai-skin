# Versioning

`yai-skin` follows Semantic Versioning: `MAJOR.MINOR.PATCH`.

- `MAJOR`: breaking changes to public skin contract (tokens, selectors, entrypoint behavior).
- `MINOR`: backward-compatible additions (new tokens, components, themes).
- `PATCH`: backward-compatible fixes and refinements.

The canonical current version is stored in `VERSION` and mirrored in `package.json`.
